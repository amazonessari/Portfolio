# Portfolio Site

This is my portfolio site that I designed from scratch

## Getting Started

It is built by open-source static site generators HUGO.

### Install Hugo

```
brew install hugo
```

To verify your new install:

```
hugo version
```

## Running the tests

### Run Hugo server

```
hugo server
```
